Body Mass Index Calculator

Body mass index (BMI) is a measure of body fat based on height and weight that applies to adult men and women. Body Mass Index (BMI) is the weight (kg) divided by the height (m) squared.

BMI = weight / height^2

This Webapp is built on top of a tiny and lightweight Python web framework that offers valuable tools and features that make developing web apps in Python relatively clear and straightforward. Project Possible's team designed and developed this dynamic web application with computation functionality within  frontend of this Web application using HTML and CSS, while the Backend was created using a third-party Python library, and the computations were completed using the Python programming language. Our team chose AWS for the deployment and hosting of our web application because of it's reliability , service , customer support and end to end availability of all the services which we required for our project. AWS also provides detailed comprehensive documentation to learn.
